# Ivanti Neurons Interview Hub

Static GitHub Pages interview-prep platform for Ivanti-focused roles.

## Included
- role-driven interview paths
- module pages
- topic pages with definition, why it matters, types/patterns, interview questions, related coding, use cases, and tricky questions
- coding and use case detail pages
- quiz with next / previous / exit
- bookmarks with local storage
- dark mode toggle
- search
- social preview image
- Google Analytics placeholder via `window.IVANTI_GA_ID`

## Deploy
Upload the contents of this folder so `index.html` is at the repository root, then publish with GitHub Pages from `main` / root.
